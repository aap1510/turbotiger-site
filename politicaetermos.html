Política de Privacidade

Atualizado em: 01/10/2025

Esta Política de Privacidade descreve como a Turbo Tiger coleta, usa, armazena e compartilha dados pessoais dos usuários do aplicativo e do site. Ao utilizar nossos serviços, você concorda com as práticas descritas nesta política.

1. Controlador

O controlador dos dados é Turbo Tiger. Para questões relativas à privacidade, entre em contato pelo e-mail: comunidade@turbotiger.com.br.

2. Dados coletados

Podemos coletar os seguintes dados pessoais:

  • Nome, e-mail, telefone;
  • Dados de cadastro (login, identificador indicador);
  • Dados técnicos do dispositivo (modelo, versão Android, ID gerado pelo app);
  • Dados de utilização (ações no app, preferências) para melhorar o serviço;

3. Finalidade do tratamento

Os dados são usados para:

  • Registrar e identificar usuários;
  • Associar indicador quando aplicável;
  • Prestar suporte e melhorar o aplicativo;
  • Cumprir obrigações legais e de segurança.

4. Base legal

O tratamento é realizado com base na execução do contrato (uso do app), no consentimento do titular quando necessário e em interesses legítimos para prevenção de fraude e melhorias do serviço.

5. Compartilhamento de dados

Não comercializamos dados pessoais. Podemos compartilhar informações com provedores de infraestrutura (AWS/Render/GitHub/Netlify) e serviços terceirizados estritamente necessários para a operação do app.

6. Segurança

Adotamos medidas técnicas e administrativas para proteger dados contra acesso não autorizado, alteração, divulgação ou destruição. Nenhuma transmissão pela Internet é 100% segura; no entanto, aplicamos boas práticas (HTTPS, variáveis de ambiente, tokens).

7. Retenção

Conservamos dados pelo tempo necessário às finalidades informadas, ou conforme exigido por lei. Dados de indicação (indicador + token) expiram automaticamente em 30 dias por padrão.

8. Direitos do titular

Você pode solicitar acesso, retificação, anonimização, portabilidade ou eliminação dos dados pessoais, além de retirar o consentimento quando aplicável. Envie pedido para comunidade@turbotiger.com.br.
